Hausaufgabe 15: Git

2.1. Fassen Sie in Ihren eigenen Worten den git-Workflow zusammen.

Auf Github wird das gewünschte Repository "geforkt" also festgemacht. Anschließend kann der Link gedownloadet/Kopiert werden und mit dem Befehl git clone in das Verzeichnis in dem man sich gerade befindet kopiert bzw. geklont werden. Durch git add werden Dateien, die im Repository hinzugefügt oder geändert werden sollen vorbereitet für die Zwischenspeicherung welche aber noch nicht erfolgt.
Wird im Anschluss git commit verwendet, werden die Dateien erst zwischengespeicher.
Durch git push werden alle Änderungen ins Repository hochgeladen und möchte man sein Arbeitsverzeichniss mit dem Repository syncronisieren, wird git pull verwendet.

2.2. Lassen Sie via ein Tooltipster-Tooltip Beispiel-Befehle bei Erwähnung der unter 2.1 erwähnten Begriffe anzeigen

Gehe über die fettgedruckten Wörter und lass dir die Beispiel-Befehle anzeigen!

$ git clone https://github.com/Cline84/EWP-HA-15.git
$ git add index.html
$ git commit -m "Änderung der index.html Datei"
$ git push origin master
$ git pull
